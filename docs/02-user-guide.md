# Panduan Pengguna

Panduan ini menjelaskan cara menggunakan OPTIMA untuk aktivitas audit sehari-hari.

---

## Login

1. Buka OPTIMA di browser
2. Masukkan email dan password yang terdaftar
3. Klik "Sign In"

Jika belum punya akun, hubungi administrator sistem melalui link di halaman login.

---

## Menu Utama

Setelah login, kamu akan melihat sidebar dengan bagian-bagian berikut:

### Main

- **Dashboard** - Ringkasan statistik dan grafik audit
- **Manager** - Tampilan performa tim (khusus manager)

### Resources

- **Regulations** - Kebijakan dan prosedur perusahaan
- **Tutorials** - Panduan dan konten how-to
- **Branch Directory** - Informasi cabang lengkap

### Audit Tools

- **Update Audits** - Input dan update data audit
- **QA Workpapers** - Review dokumentasi audit
- **Assignment** - Buat surat penugasan
- **Tools** - Tools pemrosesan tambahan

### Communication

- **Broadcast** - Kirim pengumuman
- **History** - Lihat notifikasi sebelumnya

### Admin

- **User Control Panel** - Kelola pengguna (superadmin only)

---

## Tugas Umum

### Melihat Dashboard

Dashboard menampilkan:

- Total cabang yang diaudit
- Ringkasan kasus fraud
- Komposisi audit (regular vs special)
- Tren bulanan
- Breakdown per regional
- Peta cabang interaktif

Gunakan filter untuk mempersempit berdasarkan periode atau regional.

---

### Update Data Audit

1. Buka **Update Audits**
2. Cari audit kamu di tabel
3. Klik ikon edit
4. Isi kolom yang diperlukan
5. Simpan perubahan

Sistem akan mencatat siapa yang mengubah dan kapan.

---

### Download Surat Penugasan

1. Buka **Assignment**
2. Pilih audit
3. Klik "Generate Letter"
4. Download atau print PDF

---

### Cek Informasi Cabang

1. Buka **Branch Directory**
2. Cari berdasarkan nama atau kode cabang
3. Klik cabang untuk melihat detail
4. Gunakan tampilan peta untuk melihat lokasi

---

## Sesi dan Logout

### Logout Otomatis

Untuk keamanan dan performa server, OPTIMA akan otomatis logout setelah **60 menit tidak aktif**.

Kamu akan menerima peringatan **5 menit sebelum** sesi berakhir.

Aktivitas yang mencegah logout:

- Klik di mana saja
- Mengetik
- Scroll
- Berinteraksi dengan form

### Logout Manual

Klik profil kamu di pojok kiri bawah dan pilih "Sign Out".

---

## Tips

- Bookmark dashboard untuk akses cepat
- Cek bell notifikasi secara rutin untuk update
- Gunakan fungsi search untuk mencari cabang atau audit dengan cepat
- Jika ada yang terlihat salah, gunakan tombol "Report Error" di Tools
