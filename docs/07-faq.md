# Pertanyaan yang Sering Diajukan

Pertanyaan umum tentang penggunaan OPTIMA.

---

## Login dan Akses

### Kenapa saya tidak bisa login?

Cek hal-hal berikut:

- Alamat email benar
- Password benar (case-sensitive)
- Caps Lock tidak aktif
- Akun kamu sudah dibuat oleh admin

Jika masih tidak bisa login, hubungi IT untuk reset password.

---

### Kenapa saya logout otomatis?

OPTIMA otomatis logout setelah **60 menit tidak aktif** untuk keamanan dan performa server. Kamu akan menerima peringatan 5 menit sebelumnya.

Untuk tetap login, cukup berinteraksi dengan halaman (klik, ketik, scroll).

---

### Kenapa muncul "Access Restricted"?

Ini berarti:

- Kamu tidak login, atau
- Sesi kamu sudah expired

Solusi: Klik "Login to Dashboard" dan login lagi.

---

## Data dan Dashboard

### Kenapa data saya tidak muncul?

Kemungkinan penyebab:

- Sync data belum selesai
- Kamu memilih rentang tanggal tanpa data
- Filter mengecualikan data

Coba sesuaikan filter atau tunggu siklus sync berikutnya.

---

### Kenapa cabang tidak ada di daftar?

Data cabang disinkronkan dari core system setiap hari. Jika cabang baru saja dibuat, mungkin perlu waktu hingga 24 jam untuk muncul.

Jika cabang seharusnya ada, hubungi IT.

---

### Kenapa angkanya berbeda dari catatan saya?

OPTIMA menampilkan data sesuai waktu sync terakhir. Jika kamu baru melakukan perubahan, mungkin belum tercermin sampai sync berikutnya atau setelah refresh.

Pastikan juga rentang tanggal dan filter yang sama.

---

## File dan Download

### Kenapa file saya hilang?

File yang didownload bersifat temporal dan terikat dengan sesi:

- File tidak disimpan permanen
- Logout menghapus download
- Timeout sesi menghapus file

Selalu download file segera setelah generate.

---

### Kenapa PDF tidak mau generate?

Kemungkinan penyebab:

- Browser memblokir popup
- Ukuran data terlalu besar
- Timeout jaringan

Coba:

- Izinkan popup dari OPTIMA
- Kurangi rentang data
- Refresh dan coba lagi

---

## Performa

### Kenapa OPTIMA lambat?

Dataset besar atau chart kompleks dapat mempengaruhi performa. Coba:

- Persempit rentang tanggal
- Kurangi filter
- Tunggu load awal selesai
- Clear cache browser

---

### Kenapa peta lama loadingnya?

Peta cabang memuat banyak data point. Load pertama lebih lama karena data sedang di-cache. Kunjungan berikutnya akan lebih cepat.

---

## Fitur

### Bagaimana cara generate surat penugasan?

1. Buka menu Assignment
2. Pilih audit
3. Klik Generate Letter
4. Download PDF

---

### Bagaimana cara update informasi audit?

1. Buka Update Audits
2. Cari audit kamu di tabel
3. Klik tombol edit
4. Lakukan perubahan
5. Simpan

---

### Bagaimana cara melihat jadwal audit saya?

Buka Account Settings dan cek tab Audit Schedule.

---

## Lainnya

### Bagaimana cara ganti password?

1. Buka Account Settings
2. Klik Change Password
3. Masukkan password lama dan baru
4. Simpan

Atau gunakan "Forgot Password" di halaman login.

---

### Bagaimana cara melaporkan masalah?

1. Buka Tools
2. Klik "Report Error"
3. Jelaskan masalahnya
4. Submit

Kamu juga bisa menghubungi IT langsung.

---

### Siapa yang harus dihubungi untuk bantuan?

Untuk masalah teknis: IT Help Desk

Untuk permintaan akses: Manager atau IT Admin

Untuk pertanyaan data: Tim QA
